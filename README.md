# VodSync

World of Warcraft addon that records fight timestamps for syncing your stream VOD with your raid logs on [Warcraft Logs](https://www.warcraftlogs.com).

## Install

- **CurseForge**: (TBD)
- **WoWUp**: (TBD)
- **Manual**: copy this folder to `World of Warcraft/_retail_/Interface/AddOns/VodSync/`

## What it does

While you raid, the addon writes fight start/end timestamps. After the raid, the data is used by [vodtool-web](https://github.com/ChipsetZ/vodtool-web) (or a compatible viewer) to align your VOD with the WCL combat log timeline.

## Files

- `VodSync.toc` — addon manifest
- `VodSync.lua` — main logic
